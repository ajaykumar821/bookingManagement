# bookingManagement

## Tech Stack

### Backend
- Node.js
- Express.js
- MySQL

### Frontend
- React.js
- Tailwind CSS

## Setup Steps

### Clone the repository
git clone https://github.com/ajaykumar821/bookingManagement.git

### Setup Backend
cd backend
npm install
Create a MySQL database named booking_db
Create table bookings
Update your MySQL credentials in db.js
Start backend server (node server.js)

### Setup Frontend
cd ../frontend
npm install
npm start
